# 项目格式化，初始化脚手架

# 安装

npm install -g tools

## 使用

```
Usage: tools [options] [command]

Options:
  -h, --help                display help for command

Commands:
  update                    升级应用
  pr                        提pr功能
  version                   应用版本
  help [command]            display help for command
```
